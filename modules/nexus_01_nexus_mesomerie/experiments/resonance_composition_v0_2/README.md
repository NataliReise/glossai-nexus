# Resonance Composition V0.2 Microprototype

This directory contains an isolated poetic microprototype for the redesigned Nexus 01 Return Resonance composition.

It does not replace or modify the V0.1 production renderers.

## Purpose

The prototype tests whether the selected Chamber elements can shape a substantially more singular poem without placing their current fixed formulations directly into the visible output.

```text
Chamber selections
  -> poetic profiles
  -> bounded operator space
  -> one local composition decision
  -> newly written long form
  -> linked cryptic Nexus Echo
```

The earlier curated texts remain lineage and design evidence. They influence the profiles but are not required to appear as quotations, phrases, or complete lines.

## Three prototype cases

### Case A - canonical origin world

```text
image                waiting-lantern
image response       appearing-path
scent                summer-rain
scent response       possibility-of-encounter
movement             falling-feather
movement response    crossing-feather
completion mode      crossing-encounter
```

This case tests whether a familiar V0.1 source world can produce visibly new language while retaining waiting, emerging passage, gentle atmosphere, and encounter.

### Case B - mixed threshold and return world

```text
image                open-starry-window
image response       answering-distant-light
scent                summer-rain
scent response       possibility-of-encounter
movement             returning-tide
movement response    stream-back-to-sea
completion mode      converging-return
```

This case tests a mixed combination for which the exact V0.1 Echo-path renderer has no complete canonical path. It should combine threshold, distance, soft trace, return, and convergence without quoting the old formulations.

### Case C - interior release world

```text
image                book-bench
image response       two-voices-one-page
scent                books-and-cedar
scent response       open-books-beside-one-another
movement             loosening-knot
movement response    gathering-without-tightening
completion mode      released-connection
```

This case tests an interior and relational world shaped by reading, memory, room, loosening, and connection without renewed constraint.

## Prototype stages

### Stage 1 - profile extraction

Describe each selected element through small curated fields such as:

```text
motifs
spatial affordances
atmospheric qualities
temporal qualities
movement qualities
operator affinities
avoidances
```

No profile contains an interpretation of the free words.

### Stage 2 - long-form composition library

Create a small set of reviewed sentence and line forms that:

- accept any valid free word as a capitalised poetic figure;
- require no inflection or semantic classification of that word;
- use the active profiles rather than old visible lines;
- make image, scent, movement, and continuation perceptibly relevant;
- preserve the completion mode;
- avoid unrestricted word assembly.

### Stage 3 - shared Composition Plan

Select one inspectable plan containing, at minimum:

```text
active profile influences
completion mode
primary relational operator
secondary operator or atmospheric action
selected long-form structures
selected visible motifs
selected lexical trace for the Echo
wish-word position in the Echo
```

Production randomness may be fresh and local. Prototype randomness should be injectable or seedable so that examples and failures can be reproduced.

### Stage 4 - cryptic Nexus Echo

Derive the Echo from the completed long form rather than independently from the Return Artifact.

The Echo must:

- contain exactly five lines;
- follow `2 / 4 / 6 / 4 / 1` words;
- contain the capitalised wish word exactly once in line 2, 3, or 4;
- contain the capitalised return word exactly once as line 5;
- inherit at least one sensory motif from the long form;
- retain at least one movement or relational operator;
- inherit at least one concrete lexical fragment from the long form;
- remain punctuation-free;
- behave as a relational riddle rather than a summary.

## Free-word contract used by the prototype

Wish word and return word follow the same provisional rule:

```text
one non-empty word
letters only
no whitespace
no hyphens
no digits
no punctuation or quotation marks
no part-of-speech restriction
```

The submitted lexical value is preserved as input identity. The visible poetic form begins with an uppercase letter without otherwise intentionally rewriting the word.

The exact Unicode-aware validator remains a prototype implementation question.

## What the prototype should measure

Human review should examine:

- variety across repeated runs;
- coherence without over-explanation;
- perceptible influence of every selected profile family;
- fidelity to the movement continuation and completion mode;
- centrality of wish and return words;
- productive rather than arbitrary strangeness;
- traceable relationship between long form and Echo;
- frequency of repeated templates or conspicuous phrases;
- whether the old deterministic wording has genuinely moved behind the surface.

Structural tests should verify:

- valid input handling;
- no empty compatible pool for any of the three cases;
- word-neutral use of the free words;
- completion-mode compatibility;
- mandatory Echo linkage conditions;
- exact Echo word counts;
- reproducible results under injected test randomness;
- more than one valid output per case where the pool permits.

## Boundary

This experiment must not:

- import itself into the production opener;
- change the V0.1 language library;
- move archive candidates;
- change the Return Artifact schema;
- publish or transmit any result;
- introduce AI or semantic services;
- claim complete 125-combination coverage.

Its first task is narrower:

```text
Can three carefully profiled resonance worlds
produce varied coherent long forms
and linked cryptic Echoes
without reusing the old visible formulations
```
