# Manual Pairs for the Mixed Threshold and Return World V0.1

This document contains five manually curated long-form and Nexus Echo pairs for the second poetic microprototype world.

They are **not** production outputs and should not become fixed complete paths.
They are a language laboratory for identifying reusable structures, profile influences, relational operators, and Echo-linkage rules before a local composer is implemented.

## Active prototype world

```text
image                open-starry-window
image response       answering-distant-light
scent                summer-rain
scent response       possibility-of-encounter
movement             returning-tide
movement response    stream-back-to-sea
completion mode      converging-return
wish word             Hope
return word           Spirit
```

The visible language below should be newly composed.
The old V0.1 formulations influence the underlying profiles but are not quoted as required source text.

The completion mode must preserve two directions returning toward relation without one word replacing or dissolving into the other.

## Pair 1 — One shore without fusion

### Main structural experiment

```text
threshold
-> distant signal
-> two returning waters
-> shared direction without merger
```

### Long-form Resonance Artifact

```text
At the open edge of night
rain leaves a silver trace across the floor

Hope stands where the room gives way to distance
one light remains awake beyond the glass

Below it the tide gathers its scattered roads
a narrow current answers from the dark

they return toward one shore without becoming one
Spirit waits in the water between them
```

### Nexus Echo

```text
Night threshold
one light remains awake
Hope follows waters toward one shore
rain keeps the distance
Spirit
```

### Echo check

```text

relational movement    follows toward return convergence
lexical inheritance    one light remains awake
```

### Observation target

This pair tests whether convergence can feel complete without being represented as fusion or final arrival.

## Cross-pair observations

### Structures that appear promising

The five pairs suggest several potentially reusable structural families:

```text
threshold establishes the field
sensory atmosphere changes how the field is perceived
wish word enters as a figure or direction
movement begins or returns
response introduces a second direction
completion appears as crossing convergence interval or shared orientation
return word occupies the remaining relational space
```

These are structural functions rather than fixed stanza positions.
The future composer should be able to vary their order where compatibility permits.

### Promising relational operators

```text
following toward
crossing
answering across distance
touching before meeting
sharing direction
holding an interval open
returning without undoing
carrying without absorbing
remaining at the point of relation
```

### Important avoidances confirmed by the examples

```text
Hope does not become Spirit
Spirit does not solve Hope
one current does not erase the other
a shared shore does not require fusion
return does not undo departure
meeting does not require closure
```

### Echo linkage finding

The inherited fragment can range from a complete Echo line to a shorter lexical core.
For this first set the clearest results come from an exact four-word or six-word sequence already present in the long form.

That is not yet a final rule.
Later examples should test whether a shorter characteristic verb phrase can create enough perceptible lineage.

### Echo variation finding

The fixed `2 / 4 / 6 / 4 / 1` form still permits meaningful variation through:

```text
wish position in line 2 3 or 4
sensory noun group in line 1
literal versus abstract movement in line 3
trace consequence in line 4
different degrees of syntactic openness
```

This set currently uses wish positions 2 and 3.
A later set should deliberately test line 4 without making the final two lines feel mechanically repetitive.
